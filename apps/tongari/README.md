# 🪄 とんがり帽子風 魔法ドローアプリ（プロトタイプ）

『とんがり帽子のアトリエ』の世界観にインスパイアされた、スマホで魔法陣を描いて遊ぶ非公式のデジタル玩具（プロトタイプ）です。
クエストやクリア目標はなく、「魔法を描く手触り」と「それに応じた世界のリアクション」を楽しむインタラクティブなアプリとして構築します。

---

## 📱 アプリケーション概要

- ターゲットプラットフォーム: スマートフォン（Android / iOS）
- 画面向き: 縦画面固定
- 開発環境: Godot Engine (GDScript)

---

## 🎮 コアゲームプレイループ

[ 画面をタッチして魔陣を描く ]
           │
           ▼
[ 指を離すと、形を判定 ]
           │
 ┌─────────┴─────────┐
 ▼ (成功: 紋章を認識)   ▼ (失敗: 判別不可)
[ 属性エフェクト発動 ]   [ 魔陣が消滅 ]
[ ふでむしが反応 ]      [ ふでむしが首を傾げる ]

---

## 🛠️ 機能要件

### 1. メイン画面（キャンバス）
- 描画エリア: 画面の大部分を占める半透明、または無地のキャンバス。
- ふでむし配置エリア: 画面の隅（あるいはキャンバスの背景）に、常駐するふでむしの2Dスプライトを配置。

### 2. タッチ＆ドローシステム（Line2D）
- 画面をなぞると、指の軌跡に合わせて滑らかな線（インク風のエフェクト）がリアルタイムに描画される。
- 指を離す（タッチ終了）までを一筆の「魔陣」として扱う。

### 3. 魔陣認識システム（ジェスチャ判定）
- 指が離れた瞬間に、描かれた座標配列をリサンプリング（点を等間隔に正規化）。
- あらかじめ登録した基本パターン（○・△・□）と照合し、最も近似している「属性」を決定する。
- どのパターンにも似ていない、または線が短すぎる場合は「失敗」とする。

---

## 🪄 インタラクション仕様（属性とリアクション）

| 描いた形（認識） | 発動する魔法（属性） | 画面エフェクト（GPUParticles2D） | ふでむしのリアクション（AnimationPlayer） |
| :--- | :--- | :--- | :--- |
| **○（円形）** | **光（ルミス）** | 魔陣の軌跡がじんわり発光し、画面全体にキラキラした光の粒子が広がる。 | 眩しそうに目を細める、または嬉しそうに光をパチパチ見つめる。 |
| **△（三角形）** | **火（フラム）** | 魔陣の中心から炎の粒子が吹き上がり、パチパチと火の粉が舞う。 | 熱がって一瞬画面外に逃げる、またはお尻から煙を出す。 |
| **□（四角形）** | **水（リグ）** | 魔陣から水しぶきが上がり、波紋が広がる（スマホの傾きに合わせて少し揺れる）。 | 葉っぱの傘を差す、またはパシャパシャと飛び跳ねる。 |
| **判別不可** | **失敗（暴発なし）** | 線が粒子になってサラサラと消えていく。 | 首を傾げる、またはつまらなそうにあくびをする。 |

## 🚀 開発ロードマップ

- [ ] **Phase 1: 最小限の描画**
  - Godotで縦画面プロジェクトを作成
  - スマホで画面をなぞったら Line2D で線が描け、指を離したら消える処理の実装
- [ ] **Phase 2: エフェクトとアニメの連動**
  - ふでむしの仮画像を配置
  - 「指を離したら、線が消える代わりにパーティクルが出て、ふでむしが跳ねる」という一連のリアクションの繋ぎ込み
- [ ] **Phase 3: 形状認識の組み込み**
  - シンプルなジェスチャ認識ロジックを組み込み、○・△・□の描き分けでエフェクトとふでむしのアニメーションが切り替わるように実装
