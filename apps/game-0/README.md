縦画面・妥協なきスマホゲーム開発案
コンセプト

• 縦画面・片手操作

• 隙間時間（1戦30秒〜1分）で進められる

• 指を動かす楽しさ（操作感）× パーティ構築の深さ

操作システム（役職固定）

• フリック（弾く/連発）: 中距離連射攻撃（スカウト・ガンナー系）

• タップ（連打）: 近距離ラッシュ（ファイター・アタッカー系）

• ホールド（長押し）: 近距離強攻撃・ガード（重戦士系）

• 引っ張り（エイム）: 遠距離狙撃・大技（スナイパー・魔導師系）

ゲームサイクル

• 一人プレイ: 操作キャラを適宜切り替え（スイッチアクション）。非操作キャラはAIで自動援護。

• 単純作業: フリック連射等で敵をなぎ倒し、素材をジャラジャラ集める中毒性。

• 思考要素: 役職の組み合わせ、AIの作戦設定、コンボの組み立て。

• マルチプレイ: 非同期/リアルタイムでの協力・対戦。

技術・設計指針

• 待ち時間ゼロのレスポンス

• 物理演算とHapticsによる徹底した「手触り」の追求

• 中断・再開のシームレス化（0.5秒復帰）
